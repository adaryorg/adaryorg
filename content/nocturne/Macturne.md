# [[Macturne]]
The idea behind Nocturne is to create a variant for every environment that I work on, and even if Linux was always the primary focus, I do spend a lot of time on MacOS, primarily at my real job, and I keep tweaking it to my liking, and this weekend a big one hit me: I already have nocturne on Mac!

I have my desktop set up to very minimalistic settings as much as MacOS allows it. I have a custom tiling window 'manager', i have my shell environment set up the way I like it, I have my shortcust set up as close as I can have them to my Linux setup, and all that is left to do now is wrap it all up in a neat installation script and share it with the world!

This will probably happen over the course of the next few weeks. First i need to set up my two older macs to test all the iterations of this installation script (two for Intel and Apple silicon. Fortunately I have both sitting around). I have to identify all the brew packages that have to be installed, and see if there is any software that I have to install that is not avaiable from Brew. I also have to learna. bit more about setting up Mac from the command line. I know that most of what I need to configure is possible, but I have to dot all the I's and cross all the T's to deliver something that really works out of the box. 

More updates will come soon!
